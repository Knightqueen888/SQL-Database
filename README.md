# SQLite-Database
Ich habe mit SQLite gearbeitet. SQLite ist ein Programm, mit dem Daten in Tabellen gespeichert und organisiert werden können. Man kann sich das wie eine digitale Sammlung von Listen vorstellen, in denen Informationen übersichtlich eingetragen sind. In der verwendeten Datenbank gab es verschiedene Tabellen mit Angaben zu Kunden, Mitarbeitern, Verkaufsumsätzen und Musikgenres.

Um mit diesen Daten zu arbeiten, verwendete ich die Sprache SQL. SQL ist eine Computersprache, mit der man der Datenbank bestimmte Anweisungen geben kann. Diese Anweisungen nennt man Befehle. Mit dem Befehl SELECT können bestimmte Informationen aus Tabellen angezeigt werden, zum Beispiel alle Kunden oder bestimmte Verkaufsdaten. Der Befehl WHERE wird genutzt, um Ergebnisse zu filtern, damit nur passende Daten angezeigt werden. Mit ORDER BY können Daten sortiert werden, beispielsweise nach Namen oder Umsatzhöhe.

Das Ziel des Projektes war es, mithilfe dieser SQL-Befehle gezielt Informationen aus der Datenbank herauszusuchen, zu filtern und auszuwerten. Vereinfacht erklärt funktioniert die Datenbank wie ein großes digitales Archiv, und SQL hilft dabei, schnell genau die Informationen zu finden, die benötigt werden, ohne alle Tabellen einzeln durchsuchen zu müssen.

Purpose

Die Informationen aus der Datenbank können genutzt werden, um Daten besser zu verstehen und auszuwerten. Zum Beispiel kann man sehen, welche Musikgenres am meisten verkauft werden, welche Kunden besonders oft einkaufen oder welche Umsätze erzielt wurden. Dadurch kann man Zusammenhänge erkennen und wichtige Informationen schneller finden.

Der Nutzen davon ist, dass Entscheidungen einfacher getroffen werden können. Unternehmen können zum Beispiel erkennen, welche Produkte beliebt sind oder welche Bereiche gut laufen. Außerdem spart die Arbeit mit SQL viel Zeit, weil die benötigten Daten nicht einzeln gesucht werden müssen. Stattdessen kann man mit wenigen Befehlen genau die Informationen anzeigen lassen, die man braucht.

Das Projekt zeigt also, wie Datenbanken dabei helfen, große Mengen an Informationen übersichtlich zu organisieren und sinnvoll zu nutzen.
